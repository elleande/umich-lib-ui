---
title: Icon
---

Use this to render SVG icons.

```jsx
// @live
<Icon icon="check-circle" />
```

## Size

Adjust the pixel `size`.

```jsx
// @live
<ul className="y-spacing">
  <li>
    <Icon icon="check-circle" size={16} />
  </li>
  <li>
    <Icon icon="warning" size={32} />
  </li>
  <li>
    <Icon icon="error" size={64} />
  </li>
</ul>
```
