## Button props

|Prop|Type|Required|Default|
|----|----|--------|-------|
|[kind](#kind)|enum "default" or "primary"|false|"default"|
|[small](#small)|boolean|false|"false"|
|[children](#children)|node|true|n/a|
|[element props](#element-props)|spread|false|n/a|

### kind

*Type*: enum "default" | "primary" default: "primary"

Controls how the `Button` is styled.

```jsx live=true
<Button>Okay</Button>
```

```jsx live=true
<Button kind="primary">Okay</Button>
```

### small

*Type*: boolean default: "false"

Controls if the `Button` is styled smaller.

```jsx live=true
<Button small>Small</Button>
```

```jsx live=true
<Button kind="primary" small>Small primary</Button>
```

### children

*Type*: `node`

Regular React children.

```jsx live=true
<Button kind="primary">
  <Icon icon="search" size={24} /> Search
</Button>
```

```jsx live=true
<Button>
  <span>Anything can go here.</span>{" "}
  <span>It's true.</span>
</Button>
```

### element props

Any props not listed above will be spread onto the underlying element.

```jsx live=true
<Button
  style={{ width: '10rem' }}
  className="a-custom-class"
>
  Okay
</Button>
```
