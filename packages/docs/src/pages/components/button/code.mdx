import { Button } from '@umich-lib/core'

```jsx live=true
<Button>Example</Button>
```

## Button Props

|Prop|Type|Required|Default|
|----|----|--------|-------|
|[element props](#element-props)|spread|false|n/a|
|[kind](#kind)|enum "primary" or "secondary"|false|"primary"|
|[children](#children)|node|true|n/a|

### element Props

Any props not listed above will be spread onto the underlying element.

```jsx live=true
<Button
  style={{ width: '10rem' }}
  className="class-for-button"
>
  Primary
</Button>
```

### kind

*Type*: enum "primary" | "secondary" default: "primary"

Controls how the `Button` is styled.

```jsx live=true
<Button>Primary</Button>
```

```jsx live=true
<Button kind="secondary">Secondary</Button>
```

### children

*Type*: `node`

Regular React children.

```jsx live=true
<Button>
  <Icon icon="search" size={24} /> Search
</Button>
```

```jsx live=true
<Button>
  <span>Anything can go here</span>
</Button>
```
