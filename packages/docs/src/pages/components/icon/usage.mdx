import { COLORS } from '@umich-lib/core'

## Color

An icon inherits color.

```jsx live
<div
  style={{
    color: COLORS.teal[400]
  }}
>
  <Icon icon="check_circle" /> Yes!
  <span
    style={{
      color: COLORS.coral[500]
    }}
  >
    <Icon icon="error" />
    No!
  </span>
</div>
```
