<Text lede>Use Loading component when retrieving data or performing slow computations. It helps to notify users that loading is underway.</Text>

Although it may not be obvious what is occurring on the back-end, we can communicate clearly to reassure the user that progress is happening. It is best practice to use a loading spinner whenever the wait time is anticipated to be longer than three seconds.
